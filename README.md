# josefacafe

Julio 2022
Aldana Carioli

El sitio web JOSEFA ha sido realizado con fines académicos para el proyecto correspondiente a la Diplomatura Desarrollador Web Full Stack en UTN.
Sin embargo, también representa una contribución a la cafetería JOSEFA Café y Pan, ubicada en el barrio porteño de Flores y administrada por Mariano.

Para la realización del presente proyecto se han utilizado el editor de código Visual Studio Code, motores de plantilla Handelbars, y para el modelado de bases de datos, My SQL Workbench.
En referencia al deploy se han utilizado las herramientas GitHub, Heroku y ClearDB. A su vez, ha sido de gran ayuda para pruebas internas la aplicación Postman.
